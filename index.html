改变主题和发布博客
2023/11/2

1. 改变主题

  1. 访问 Hexo 所有的主题都在这个地址。
  2. 下载完，把它放在Blog文件夹内Blog/themes这个文件夹下
    3.在_config.yml内修改thema的名称为刚刚下载的

2. 发布博客

1.在gitbash命令下，写hexo new “博客名”
2.在Blog/source/_posts下存放着发布的博客，点击修改就行

最后不管更改了什么，直接在bash命令下无脑运行这三个命令

hexo clean
hexo generate
hexo deploy

Hexo 博客部署到 GitHub
2023/10/5

以下是具体步骤：

1. 创建 GitHub 仓库

  1. 访问 GitHub 并登录您的帐户。
  2. 创建一个新的仓库。仓库名称应该是 <your-username>.github.io，其中 <your-username> 是您的 GitHub 用户名。

2. 下载Nodejs和Git

1.成功配置好Nodejs后，可以利用Git Bash进行测试
分别执行一下命令，查询版本号

$ node -v
$ npm -v

3. 安装和配置 Hexo

1.在git bash执行下面的命令安装hexo（例如在F盘新建了一个blog文件夹，就在blog文件夹下打开git bash）

cnpm install hexo-cli -g
  1. 打开您的 Hexo 博客目录。
  2. 打开 _config.yml 文件。
  3. 在文件的底部，添加或修改 deploy 部分，使其指向您刚刚在 GitHub 上创建的仓库。它应该看起来像这样：
deploy:
  type: git
  repo: https://github.com/<your-username>/<your-username>.github.io.git
  branch: main

然后请将 <your-username> 替换为您的 GitHub 用户名。

5.初始化博客（运行这条命令需要确定Git Bash运行在Blog文件夹下）

hexo init

4. 安装 Hexo-deployer-git

如果您还没有安装 hexo-deployer-git，您需要安装它来帮助您将博客部署到 GitHub。在终端中，运行以下命令：

npm install hexo-deployer-git --save

或者如果您使用 cnpm：

cnpm install hexo-deployer-git --save

6.预览博客

hexo s

5. 部署博客到Github

现在，您可以使用以下命令将您的博客部署到 GitHub：

hexo clean
hexo generate
hexo deploy

这将构建您的博客并将其推送到您在 _config.yml 文件中指定的 GitHub 仓库。

6. 访问您的博客

一旦部署完成，您可以通过访问 https://<your-username>.github.io 来查看您的博客。

请记得替换 <your-username> 为您的 GitHub 用户名。
